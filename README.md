# Estudo de Manipulação de Arquivos em Python

Ei, galera! Sou o Edson, de Recife, e tô mergulhando no Python pra automações e projetos reais. Esse repo é um script básico que eu criei pra praticar:

- Criar e gravar dados em TXT
- Adicionar linhas novas (append)
- Ler linha a linha ou o arquivo inteiro
- Gravar uma lista simples

O código tá "cru" mesmo – com caminhos hardcoded pro meu PC Windows, partes comentadas e uns errinhos de digitação. Não é perfeito, mas é honesto pro processo de aprendizado. Se quiser testar:

1. Ajuste o caminho 'C:/PythonSAIDA.txt' pro seu setup.
2. Rode com `python manipulacao_arquivos.py` (ou o nome que der).
3. Veja o arquivo SAIDA.txt gerado!
